# xin-offer Product

## 项目一句话介绍.

xin-offer是一个面向求职者的AI模拟面试系统,帮助用户根据目标岗位进行结构化的面试练习,并获得反馈报告.

## 目标用户.

- 准备找工作的程序员.
- 想练习技术面试的程序员.
- 缺少真实面试机会的求职者.

## v0.1核心功能.

1. 用户选择目标岗位,技术栈,工作年限和面试难度.
2. AI面试官生成第一个面试问题.
3. 用户回答问题.
4. AI根据用户的回答继续追问.
5. 最多进行3轮问答.
6. 面试结束后生成反馈报告.

## v0.1暂时不做.

- 登录注册.
- 简历上传.
- 语音面试.
- 数字人形象.
- 支付.
- 管理后台.
- Milvus.
- 复杂Agent编排.

## v0.1成功标准.

用户可以完成一场3轮文字模拟面试,并获得一份包含评分,优点,不足和学习建议的反馈报告.
